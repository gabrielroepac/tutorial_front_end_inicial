📄 HTML

A estrutura básica do HTML

Todo documento HTML segue uma estrutura padrão. Ela é como o esqueleto de uma página: define onde ficam as informações que o navegador exibe e as instruções que ficam ocultas.

O que é HTML?

HTML (HyperText Markup Language) é a linguagem que define o conteúdo e a estrutura de uma página web. Tudo que você vê em um site — textos, títulos, imagens — é criado com HTML.

No HTML, utilizamos tags para organizar o conteúdo. Uma tag é composta por um nome entre sinais de menor e maior: <tag>. A maioria das tags possui abertura e fechamento:

Toda tag HTML possui uma abertura <tag>, um conteúdo e um fechamento </tag>

A estrutura completa

Cole o código abaixo no seu arquivo index.html no StackBlitz. Este é o ponto de partida do nosso blog de filmes:

index.html HTML
<!DOCTYPE html> <html lang="pt-BR"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>Meu Blog de Filmes</title> </head> <body> <header> <h1>🎬 Meu Blog de Filmes</h1> <p>Dicas e recomendações para todos os gostos</p> </header> <main> <h2>Filme em Destaque</h2> <p class="autor">Por: Maria Souza</p> <p>Uma viagem épica pelo espaço que desafia os limites da ciência e do amor.</p> </main> </body> </html>

A estrutura HTML é organizada de forma hierárquica — tags dentro de outras tags

🧩
O que cada tag faz

<!DOCTYPE html> — informa ao navegador que é um documento HTML moderno.
<head> — contém informações ocultas, como o título da aba e configurações.
<body> — tudo o que o usuário vê na página fica aqui.
<header> — define o cabeçalho visual da página.
<main> — delimita o conteúdo principal da página.

📄 HTML

Tags de texto e formatação

O HTML oferece diversas tags para organizar e formatar textos. Cada uma tem uma função semântica: não apenas muda a aparência, mas também informa ao navegador o papel daquele conteúdo.

As tags de bloco ocupam uma linha inteira; as tags inline formatam apenas parte do texto

Atualize seu arquivo index.html

Dentro do <main>, enriqueça o conteúdo do filme adicionando ênfase ao título e ao gênero:

index.html — trecho do <main> HTML
<main> <h2>Filme em Destaque</h2> <p class="autor">Por: <strong>Maria Souza</strong></p> <p> <em>Interestelar (2014)</em> — Uma viagem épica pelo espaço que desafia os limites da ciência e do amor. <br> Indicado para quem aprecia <strong>ficção científica</strong> profunda. </p> </main>
💡
Semântica importa

Use <strong> para indicar importância real do conteúdo (não apenas para deixar negrito). Use <em> para indicar ênfase (não apenas itálico decorativo). O Google e os leitores de tela entendem essa diferença.

🎨 CSS

Estilizando com CSS

O CSS (Cascading Style Sheets) é a linguagem responsável pela aparência visual da página. Com ele, definimos cores, fontes, tamanhos e disposição dos elementos.

Como o CSS funciona

O CSS funciona através de regras. Cada regra é composta por um seletor (qual elemento estilizar) e um bloco de declarações (quais estilos aplicar):

Uma regra CSS: seletor define o quê estilizar; propriedade e valor definem como

Adicionando estilos ao nosso blog

Por boas práticas de programação, não devemos misturar código HTML e CSS no mesmo arquivo. Em vez disso, vamos criar um arquivo separado chamado styles.css para guardar todos os nossos estilos.

Primeiro, dentro do <head> do seu arquivo index.html, adicione a tag <link> abaixo para fazer a ligação entre o HTML e a nossa folha de estilos externa:

index.html — ligando o arquivo CSS HTML
<head> <!-- ... outras tags meta ... --> <title>Meu Blog de Filmes</title> <link rel="stylesheet" href="styles.css"> </head>

Agora, crie um novo arquivo chamado styles.css na mesma pasta do seu projeto e adicione as regras CSS abaixo nele (não use a tag <style> aqui, pois esse já é um arquivo CSS puro):

styles.css CSS
header { background-color: #1a2744; color: #ffffff; text-align: center; max-width: 800px; margin: 0 auto; padding: 16px; } main { background-color: #ffffff; color: #1a2744; text-align: center; max-width: 800px; margin: 0 auto; padding: 16px; }
⚠️
Contraste de cores

A combinação de cores deve garantir boa legibilidade. A proporção mínima recomendada é de 4,5:1 entre texto e fundo. O azul-marinho #1a2744 com texto branco #ffffff oferece excelente contraste.

Propriedades CSS essenciais

Referência rápida

🔵background-color — cor de fundo do elemento
🔵color — cor do texto
🔵text-align — alinhamento do texto (left, center, right)
🔵max-width — largura máxima do elemento (ex: 800px)
🔵margin: 0 auto — centraliza o elemento horizontalmente
🔵padding — espaçamento interno entre o conteúdo e a borda
🎨 CSS

Box model e imagens

Cada elemento HTML é tratado como uma "caixa" pelo navegador. Compreender esse modelo é fundamental para controlar espaçamentos e bordas. Também aprenderemos a inserir imagens de forma acessível.

O modelo de caixa (Box Model)

Todo elemento HTML possui quatro camadas ao seu redor. Da parte mais interna para a mais externa:

O box model: cada elemento é uma caixa com conteúdo, espaçamento interno, borda e espaçamento externo

Adicionando uma imagem ao blog

A tag <img> insere imagens na página. Ela possui dois atributos essenciais:

  • src — o caminho ou endereço da imagem
  • alt — texto alternativo que descreve a imagem (essencial para acessibilidade)

Adicione o código abaixo dentro do <main>, antes do <h2>:

index.html — imagem e espaçamentos HTML
<main> <img src="https://upload.wikimedia.org/wikipedia/en/b/bc/Interstellar_film_poster.jpg" alt="Pôster do filme Interestelar: nave espacial sobre um planeta alienígena" > <h2>Filme em Destaque</h2> <!-- ... restante do conteúdo ... --> </main>

Agora, no CSS, controle o tamanho da imagem e adicione bordas e padding ao header:

styles.css — adicionar estilos da imagem CSS
img { width: 120px; height: 170px; border: 3px solid #e8a020; border-radius: 8px; } header { /* adicionar ao bloco existente: */ padding: 24px 16px; }
♿
Por que o atributo alt é obrigatório

O atributo alt é utilizado por leitores de tela para descrever imagens a pessoas com deficiência visual. Também é exibido quando a imagem não carrega. Sempre descreva o conteúdo relevante da imagem.

🎨 CSS

Flexbox e classes CSS

O Flexbox é um sistema de layout que permite posicionar elementos lado a lado de forma flexível. As classes CSS permitem aplicar estilos específicos a elementos escolhidos, sem afetar todos os demais.

O problema sem Flexbox

Por padrão, elementos HTML como <div>, <p> e <img> ficam empilhados verticalmente (um abaixo do outro), ocupando toda a largura disponível. Isso é chamado de display: block.

O display: flex permite colocar a imagem e o texto lado a lado

Organizando o card do filme com div e Flexbox

Primeiro, no HTML, envolva a imagem e os textos do filme em uma <div>. Isso nos permite aplicar Flexbox apenas nesse grupo:

index.html — main atualizado HTML
<main> <img src="https://upload.wikimedia.org/wikipedia/en/b/bc/Interstellar_film_poster.jpg" alt="Pôster do filme Interestelar" > <div> <h2>Filme em Destaque</h2> <p class="autor">Por: <strong>Maria Souza</strong></p> <p>Uma viagem épica pelo espaço...</p> </div> </main>

Agora, no CSS, aplique display: flex ao <main> e estilize a classe .autor:

styles.css — atualizar main e adicionar .autor CSS
main { background-color: #ffffff; color: #1a2744; max-width: 800px; margin: 0 auto; padding: 24px 16px; display: flex; /* ← posiciona img e div lado a lado */ gap: 20px; /* ← espaço entre os itens */ align-items: flex-start; /* ← alinha pelo topo */ text-align: left; } .autor { /* ← seletor de classe com ponto */ font-weight: bold; color: #888888; font-size: 13px; }
🏷️
Classes CSS: como funcionam

No HTML, atribuímos uma classe com class="nome-da-classe". No CSS, referenciamos essa classe com um ponto na frente: .nome-da-classe { ... }. O nome deve ser uma única palavra (use hífens para separar), sem espaços e sem caracteres especiais.

⚡ JavaScript

Contador de likes com JavaScript

O JavaScript é a linguagem que torna as páginas interativas. Com ele, podemos responder a ações do usuário — como um clique — e atualizar o conteúdo da página em tempo real.

HTML = esqueleto · CSS = aparência · JavaScript = comportamento

Passo 1: Adicionar o botão no HTML

Dentro do <div> que contém os textos do filme, adicione um botão com um emoji de coração e um <span> para exibir o contador:

index.html — botão de curtir HTML
<div> <h2>Filme em Destaque</h2> <p class="autor">Por: <strong>Maria Souza</strong></p> <p>Uma viagem épica pelo espaço...</p> <button>❤️ <span>0</span></button> </div>

Passo 2: Estilizar o botão no CSS

Abra o seu arquivo styles.css e adicione a regra abaixo para deixar o botão com cantos arredondados, fundo branco e espaçamento interno:

styles.css — estilo do botão CSS
button { padding: 8px 16px; border: 1px solid #ddd; border-radius: 20px; background: #ffffff; cursor: pointer; font-size: 14px; margin-top: 12px; }

Passo 3: Criar e vincular o arquivo JavaScript

Assim como fizemos com o CSS, a melhor prática é manter nosso código JavaScript em um arquivo separado chamado script.js. Isso deixa nosso HTML limpo e focado na estrutura.

Primeiro, no final do seu arquivo index.html, logo antes da tag de fechamento </body>, adicione a tag <script> vinculando o arquivo externo:

index.html — vinculando o arquivo script.js HTML
<script src="script.js"></script> </body>

Agora, crie o arquivo script.js na mesma pasta do seu projeto e adicione o código abaixo para fazer o botão funcionar:

script.js JS
// 1. Seleciona o botão no documento const botao = document.querySelector("button"); // 2. Escuta o evento de clique botao.addEventListener("click", botaoClicado); // 3. Define o que acontece ao clicar function botaoClicado() { let contador = botao.querySelector("span"); contador.textContent++; // incrementa o número em 1 }

Fluxo completo: o clique dispara o evento, que executa a função, que atualiza o número na tela

🧪
Testando o JavaScript

Pressione F12 no navegador para abrir as ferramentas de desenvolvedor. Na aba Console, você pode adicionar console.log("teste") dentro da função para verificar se ela está sendo executada corretamente.

Estado dos arquivos do projeto neste módulo

Neste ponto, o seu projeto deve estar organizado em três arquivos separados na mesma pasta:

⚡ JavaScript

Eventos em múltiplos elementos

Agora que temos um botão interativo, vamos aprender a trabalhar com vários botões na mesma página utilizando o JavaScript. Veremos como selecionar todos eles de uma vez e associar um evento a cada um usando loops.

O problema com querySelector

O método document.querySelector() seleciona apenas o primeiro elemento que corresponde ao seletor. Se nossa página tiver vários botões e tentarmos usá-lo, o JavaScript só vai controlar o primeiro botão de todos, ignorando os outros.

Para selecionar todos os botões de uma vez, precisamos usar o document.querySelectorAll(). Ele nos retorna uma lista contendo todos os elementos que correspondem ao seletor.

0 👍 0 Apenas o 1º querySelectorAll("button") ❤️ 0 👍 0 Todos (Lista)

querySelector seleciona apenas a primeira ocorrência, enquanto querySelectorAll seleciona todos os elementos em uma lista.

Passo 1: Adicionar um segundo botão no HTML

No seu arquivo index.html, adicione um segundo botão de reação dentro do <div> do filme (por exemplo, um joinha 👍):

index.html — adicionando segundo botão HTML
<div> <h2>Filme em Destaque</h2> <p class="autor">Por: <strong>Maria Souza</strong></p> <p>Uma viagem épica pelo espaço...</p> <button>❤️ <span>0</span></button> <button>👍 <span>0</span></button> </div>

Passo 2: Usar forEach para percorrer os botões

Como o querySelectorAll nos dá uma lista de botões, precisamos usar o método forEach. Ele executa uma função para cada elemento individual da nossa lista:

script.js — iterando com forEach JS
// 1. Seleciona todos os botões da página const botoes = document.querySelectorAll("button"); // 2. Percorre a lista de botões usando forEach botoes.forEach(function(botao) { // Adiciona o evento de escuta para cada botão individual botao.addEventListener("click", botaoClicado); // 3. Função que será executada ao clicar no botão específico function botaoClicado() { let contador = botao.querySelector("span"); contador.textContent++; // Incrementa apenas o span deste botão } });
🧩
Por que definir a função dentro do forEach?

Ao declarar a função botaoClicado dentro do forEach, ela ganha acesso direto à variável botao correspondente daquela iteração (um conceito chamado closure ou escopo fechado). Isso garante que o contador correto seja incrementado!

⚡ JavaScript

Lógica e limite de curtidas

Para finalizar nosso projeto, vamos adicionar um segundo post ao blog, ajustando a estrutura do layout com a tag semântica <article>. Também criaremos uma lógica para limitar as curtidas a apenas uma por post, usando booleanos e condicionais.

Criando o segundo post no blog

Um blog real possui vários posts. Vamos duplicar o conteúdo dentro do nosso <main> para criar um segundo post de filme (ex: Avatar). Porém, se fizermos isso agora com a estrutura antiga, os posts vão quebrar visualmente porque o CSS original do main possui display: flex.

Para corrigir isso, utilizaremos a tag semântica <article> para envolver cada post completo. No CSS, moveremos o comportamento do Flexbox do main para o article. Dessa forma, as imagens e textos ficarão lado a lado em cada post individual, mas os posts em si ficarão empilhados verticalmente!

CSS — Novo layout com article CSS
main { background-color: #ffffff; color: #1a2744; max-width: 800px; margin: 0 auto; padding: 24px 16px; } article { display: flex; /* ← Coloca a imagem e o texto lado a lado */ gap: 20px; align-items: flex-start; margin-bottom: 32px; /* ← Espaço entre posts */ padding-bottom: 24px; border-bottom: 1px solid #eee; }

Controlando as interações com Booleanos

Em redes sociais reais, você só pode curtir uma publicação uma vez. Se clicar novamente, a curtida é desfeita ou ignorada. Para implementar isso, vamos usar variáveis booleanas (que assumem apenas os valores true ou false) e a condicional if.

Criaremos uma variável de controle chamada curtiu, iniciada como false. Quando o usuário clica, verificamos se curtiu === false. Se for verdade, incrementamos a curtida e mudamos a variável para true. Cliques seguintes serão ignorados porque o valor agora é true!

A variável booleana de controle impede curtidas repetidas no mesmo elemento.

Código completo final com múltiplos posts e limite de likes

Abaixo está o código final do seu blog de filmes, totalmente finalizado com múltiplos posts e a lógica de curtidas limite:

index.html — Versão Final Consolidada HTML
<!DOCTYPE html> <html lang="pt-BR"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>Meu Blog de Filmes</title> <link rel="stylesheet" href="styles.css"> </head> <body> <header> <h1>🎬 Meu Blog de Filmes</h1> <p>Dicas e recomendações para todos os gostos</p> </header> <main> <!-- Post 1 --> <article> <img src="https://upload.wikimedia.org/wikipedia/en/b/bc/Interstellar_film_poster.jpg" alt="Pôster de Interestelar"> <div> <h2>Filme em Destaque: Interestelar</h2> <p class="autor">Por: <strong>Maria Souza</strong></p> <p>Uma viagem épica pelo espaço que desafia os limites da ciência.</p> <button>❤️ <span>0</span></button> <button>👍 <span>0</span></button> </div> </article> <!-- Post 2 --> <article> <img src="https://upload.wikimedia.org/wikipedia/en/d/d6/Avatar2009poster.jpg" alt="Pôster de Avatar"> <div> <h2>Outra Recomendação: Avatar</h2> <p class="autor">Por: <strong>Marcelo Paludetto</strong></p> <p>Uma jornada espetacular no mundo de Pandora, descobrindo novas culturas.</p> <button>❤️ <span>0</span></button> <button>👍 <span>0</span></button> </div> </article> </main> <script src="script.js"></script> </body> </html>
styles.css — Versão Final Consolidada CSS
header { background-color: #1a2744; color: #ffffff; text-align: center; max-width: 800px; margin: 0 auto; padding: 24px 16px; } main { background-color: #ffffff; color: #1a2744; max-width: 800px; margin: 0 auto; padding: 24px 16px; } article { display: flex; gap: 20px; align-items: flex-start; margin-bottom: 32px; padding-bottom: 24px; border-bottom: 1px solid #eee; } img { width: 120px; height: 170px; border: 3px solid #e8a020; border-radius: 8px; } .autor { font-weight: bold; color: #888888; font-size: 13px; } button { padding: 8px 16px; border: 1px solid #ddd; border-radius: 20px; background: #ffffff; cursor: pointer; font-size: 14px; margin-top: 12px; margin-right: 8px; }
script.js — Versão Final Consolidada JS
const botoes = document.querySelectorAll("button"); botoes.forEach(function(botao) { // Cada botão da lista possui sua própria variável de controle (booleana) let curtiu = false; botao.addEventListener("click", botaoClicado); function botaoClicado() { let contador = botao.querySelector("span"); // Condicional que verifica se o usuário ainda não curtiu if (curtiu === false) { contador.textContent++; // Incrementa o contador do botão curtiu = true; // Marca como curtido } } });

🚀 Deploy

Como publicar no GitHub Pages

Uma exigência do Concurso Agrinho é que seu site esteja acessível online e configurado com a tag correta. O GitHub Pages permite colocar sua página no ar gratuitamente em minutos.

Passo a Passo da Publicação

1
Crie uma conta e um Repositório
Se ainda não tiver, crie uma conta no GitHub. No seu painel, clique em New repository (Novo repositório), dê um nome ao seu projeto (ex: agrinho-2026), certifique-se de que ele seja Public (Público) e clique em Create repository.
2
Faça o upload dos arquivos
Na tela do repositório, clique em "uploading an existing file" e envie todos os arquivos do seu site (index.html, styles.css, script.js e as pastas de imagens). Role a página para baixo e clique em Commit changes.
3
Crie o arquivo README.md
O edital exige um arquivo de explicação. Clique em "Add a README" e descreva o seu projeto: qual é o tema, quais tecnologias você usou e instruções de uso. Salve clicando em Commit changes.
4
Ative o GitHub Pages
Vá até a aba Settings (Configurações) do repositório, clique em Pages (no menu lateral esquerdo). Onde diz Source (ou Branch), mude de None para main (ou master) e clique em Save. Em alguns minutos, seu site estará online em um link como https://seu-usuario.github.io/agrinho-2026/.
5
Adicione a Tag (Topic) "agrinho"
Volte para a página inicial do seu repositório (aba Code). No lado direito, perto de About, clique no ícone da engrenagem ⚙️. No campo "Topics", digite agrinho e salve. Coloque também o link do seu site finalizado no campo "Website".
⚠️
Confira antes de enviar!

Se o seu repositório estiver "Privado", os avaliadores não conseguirão ver o código nem o site. Você será desclassificado. Verifique se o link final abre corretamente no celular ou numa janela anônima.

Ideias de Temas e o Uso de IA

Está sem ideias para o tema "Agro forte, futuro sustentável" morando no meio urbano? Aqui vão algumas sugestões e dicas cruciais sobre como usar Inteligências Artificiais (como o Gemini) para te ajudar na programação.

Ideias de Projetos Urbanos

A sustentabilidade e a agricultura estão presentes no nosso dia a dia, mesmo na cidade! Aqui estão 3 ideias de sites que você pode construir:

♻️
Calculadora de Compostagem Doméstica

Um site que ensina famílias a transformar lixo orgânico em adubo. O usuário digita quantas pessoas moram na casa e o site calcula (usando JavaScript) o tamanho ideal da composteira e quanto adubo será gerado por mês.

🌿
Guia de Defensivos Naturais (Hortas Domésticas e de Quintal)

Um catálogo interativo de receitas caseiras para afastar pragas de pequenas hortas (ex: óleo de neem). O usuário escolhe o tipo de praga que tem na horta e o JavaScript filtra a receita agroecológica ideal para combater o problema.

🛒
Mapa de Feiras Orgânicas Locais

Um site focado na valorização da agricultura familiar da sua cidade. Um filtro onde o usuário seleciona o tipo de alimento (ex: hortaliças, mel) e o site mostra, usando listas no JS, em quais feiras locais encontrar os produtos.

Turbinando seu código com o Google Gemini (IA)

Você não precisa fazer tudo sozinho! Inteligências Artificiais como o Google Gemini podem ser usadas como o seu "parceiro de programação". Se você travou na lógica do JavaScript, você pode pedir ao Gemini:

  • "Me ajude a criar uma função em JavaScript que pegue o valor de um input e multiplique por 2"
  • "Meu CSS não está centralizando a div, o que pode estar errado?"
  • "Como faço para esconder um elemento da tela usando JS?"

Boas Práticas e Regras do Concurso sobre IA

🧠Entenda o código: Não copie e cole às cegas. Leia e estude a explicação que a IA te deu para aprender de verdade.
🎨Modifique: Personalize cores, textos e a estrutura do código gerado para garantir que o projeto tenha a sua própria identidade (o edital exige isso).
⚠️A Regra de Ouro (Penalização de Notas): Segundo o edital, qualquer elemento do site gerado ou alterado por IA PRECISA TER O PROMPT DESCRITO no arquivo README.md do repositório ou como comentário no próprio código. Se você usar IA e não der os devidos créditos nem descrever o que pediu para ela fazer, a sua nota no quesito "Originalidade" cairá drasticamente ou você poderá até ser desclassificado!
